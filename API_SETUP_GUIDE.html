üÜò Troubleshooting

If you get "API key not configured":

  • Make sure .env file is in fantasy-dashboard-v2/ folder (not inside src/)
  • Make sure you restarted the server after creating .env
  • Check that your key starts with sk-proj- or sk-
  • No spaces, no quotes around the key

Test in browser console (F12):

console.log(import.meta.env.VITE_OPENAI_API_KEY)

If this shows undefined, your .env file isn't loading.
